Spring Boot Mail Sender
A simple Spring Boot web application that allows users to send emails using JavaMailSender. Users can enter an email address and message body through a web form â€” and the application will send the message directly to the specified email.

Features
1) Send plain text emails
2) Simple and clean web form (HTML/CSS frontend)
3) Uses Spring Boot and JavaMailSender
4) Configurable SMTP settings via application.properties

Tech Stack
1) Java 17+
2) Spring Boot (Mail, Web)
3) Thymeleaf (or plain HTML)
4) Maven
